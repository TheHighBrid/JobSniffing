# iCIMS adapter architecture

iCIMS owns only strict platform recognition, bounded Apply, accessible iframe or SPA application-surface discovery, and optional existing-account sign-in. It does not own a page loop.

Question extraction, standard-field filling, approved custom answers, dynamic rescans, validation recovery, loop protection, screenshots, redacted journals, Next and Review traversal, final Submit separation, and confirmation detection all remain in the shared browser form and wizard layers.
